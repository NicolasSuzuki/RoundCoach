# RoundCoach Web

Frontend MVP do RoundCoach para validar o fluxo real do jogador.

## Stack

- React
- Vite
- TypeScript
- Tailwind
- React Query
- Zustand
- React Router
- Axios

## Fluxo atual

- criar conta
- login
- listar e criar partidas
- vincular VOD por URL
- disparar processamento
- acompanhar status
- abrir analise fake

## Ambiente

Copie `.env.example` para `.env`:

```env
VITE_API_URL=http://localhost:3000/api/v1
```

## Rodar localmente

```bash
npm install
npm run dev
```

A aplicacao abre em `http://localhost:5173`.
